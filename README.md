##  Setup

```sh

```


### Install Tailwind and its peer-dependencies using npm:

```sh
npm install -D tailwindcss@latest postcss@latest autoprefixer@latest
```

### Compile and Hot-Reload for Development
```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
